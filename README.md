SETUP for running tests with pytest
======

Create virtualenv with python3
------
```bash
virtualenv venv-auto --python=$(which python3)
```
Activate virtualenv
------
```bash
source venv-auto/bin/activate
```
Install requirements
------
```bash
cd ja_plp
pip install -r requirements.txt
```
